# Student Management System

## About

This is a simple Student Management System developed using Core PHP, MySQL, PDO and Bootstrap 5.

## Features

- Student Listing
- Add Student
- Edit Student
- Delete Student
- Responsive Design

## Technologies

- PHP 8.3
- MySQL
- Bootstrap 5
- PDO
- Apache 2.4

## Installation

1. Clone the repository
2. Import `student_management.sql`
3. Update database credentials in `config/database.php`
4. Run:
   http://localhost/studentmanagement/
